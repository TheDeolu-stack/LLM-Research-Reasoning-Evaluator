# Scoring Rubric
Each test case is graded using the following:


| Criteria | Score (0–2) | Description |
|---------|-------------|-------------|
| Accuracy | 0–2 | Correctness of final answer |
| Reasoning Quality | 0–2 | Step-by-step, coherent logic |
| Constraint Handling | 0–2 | Follows all instructions |
| Consistency | 0–2 | No contradictions |
| Clarity | 0–2 | Easy-to-understand explanation |


Maximum score per prompt: **10**
